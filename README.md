# Immo - Suivi d'opportunités immobilières

Suivre, noter et comparer ses opportunités immobilières.

## Setup dev environment

### 1. Run Docker

```
docker-compose up -d
```

### 2. Install dependencies

```
yarn
```

### 3. Run the server

```
yarn dev:server
```

### 4. Run the client

```
yarn dev:client
```

## Launch tests

```
yarn test
```

## Builds the app for production

```
yarn build
```
